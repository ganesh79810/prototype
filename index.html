Admin Portal
ParkSync is the proposed information system for the City of Riverton. This prototype demonstrates core functionality using a synthetic dataset of 1,200 parking citations built from real-world benchmarks. Every table, chart, and action in this system is powered by that dataset.
IS/IT Solution Prototype — MRP Spring 2026 — Saint Louis University
Riverton officers issue tickets to residents
with valid permits — because their devices show old data
The Permit Office approves permits in the central database, but officer handheld devices only receive updates every 2–3 hours via batch upload. In that gap, valid permits are invisible. Officers write tickets. Residents dispute them. City staff spend 318 hours resolving errors that should never have happened. ParkSync replaces the batch cycle with an event-driven system where approvals propagate instantly.
